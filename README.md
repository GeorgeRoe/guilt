# GUILT

Green Usage Impact Logging Tool allows users to keep track of their carbon emissions whilst working with SLURM.

You can read the docs for GUILT [here](https://GeorgeRoe.github.io/GUILT/).

## Features

- Tracks emissions from CPU usage of SLURM jobs
- Prediction of optimum time to run jobs
- Calculate emissions from previous jobs
- Produces reports of your usage which can be printed to terminal or PDF
- No admin required for install or use, can be used on any cluster using SLURM
